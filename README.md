1. **Clone the repository:**

    ```bash
    git clone https://github.com/iamswapnil22/gaadi-bazzar.git
    cd gaadi-bazzar
    ```

2. **Install dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

3. **Run the prediction web app:**

    ```bash
    cd web_app
    python app.py
    ```
